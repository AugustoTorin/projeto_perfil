# 📚 Lista de Livros - Augusto Cury

Projeto desenvolvido para praticar HTML e CSS.

A aplicação apresenta uma lista de livros do autor Augusto Cury, onde o usuário pode clicar nos títulos para ser redirecionado para páginas de compra dos livros.

## 📑 Conteúdo Estudado

- Estrutura básica do HTML;
- Criação de listas;
- Manipulação de elementos com CSS;
- Variáveis CSS;
- Organização de layout com divs.

## 📁 Estrutura do Projeto

- Pasta `css/style.css`
- Pasta `images/`
- `index.html`

## 🚀 Tecnologias Utilizadas

- HTML;
- CSS.
